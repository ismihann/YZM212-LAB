# HMM ile Ses Tanıma Ödevi

### 1. Problem Tanımı
Bu projede, Gizli Markov Modelleri (HMM) kullanarak konuşma/ses   tanıma sistemi geliştirilmiştir.

### 2. Veri
Ödev kapsamında kullanılan veriler şunlardır:
- 16 farklı fonem(kelimenin) hangi modele yakın olduğu incelenmiştir.
- Veri seti `DATA/` klasörü altında bulunmaktadır.

### 3. Yöntem
Projede şu adımlar izlenmiştir:
- Özellik çıkarımı (MFCC vb.)
- HMM modelinin eğitilmesi (`hmmlearn` kütüphanesi ile)
- Test verileri üzerinde doğruluk ölçümü.

### 4. Sonuçlar ve Yorumlar
- Yapılan testler sonucunda % [buraya başarı oranını yaz] oranında doğruluk elde edilmiştir.
